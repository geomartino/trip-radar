# Trip Radar
✈️ Carte interactive des avertissements de voyage du gouvernement du Canada

**→ https://geomartino.github.io/trip-radar/**

Affiche les niveaux de risque par pays (Affaires mondiales Canada) avec filtres, zoom/pan et liens vers les fiches détail de chaque destination. Données mises à jour automatiquement chaque nuit.
